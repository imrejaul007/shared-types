# CorpPerks Landing Page

Marketing website for CorpPerks - Smart Corporate Spend & Benefits Platform.

## Quick Start

```bash
# Install dependencies
npm install

# Start development
npm run dev

# Build for production
npm run build

# Start production
npm start
```

## Tech Stack

- Next.js 14
- React 18
- Tailwind CSS
- Lucide Icons

## Sections

- Hero with dashboard preview
- Problem/Solution cards
- Features grid
- How it Works (3 steps)
- Pricing (3 tiers)
- CTA with email capture
- Footer

## Deploy to Vercel

```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel
```

## Environment

No environment variables required for landing page.
